# Coin Count README

This project allows a user to input the amount of money they have and get the value back in the number of quarters, dimes, nickels, and pennies.

## Installation
Add the following to your gem file
```
gem('sinatra')
gem('sinatra-contrib')
gem('pry')
gem('rspec')
gem('capybara')
```
Run the following command
```
bundle install
```
